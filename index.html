The Tansey Test

Is that article or blog post really about archives? Drop a link to the full text (in HTML) in the search below and I'll tell you!

How this works

tldr; this checks a link to see how many mentions of archivists a web page contains.

It was inspired by a tweet from my friend Eira Tansey.

The longer version: reference script/fetchHtml.js
  • Use Yahoo Query Language to return a URL's HTML as a JSON object.
  • Loop through that object and return just the content as an array.
  • Loop through that array looking for a regular expression ("archivist," case-insensitive), returning the index of each match.
  • If no hits were returned, tell the user the page isn't about archives
  • If hits were returned, calculate percentage of content pieces that match the search terms, and display that for the user.
